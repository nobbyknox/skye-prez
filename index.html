Introducing the Bully Algorithm

Simply put, the lemming that shouts the loudest gets to be the leader.

Traditional Requirements

Typical implementations have certain assumptions, like peer discovery and/or specific network topologies.

Certain algorithms work with certain network topologies. Your choice is dictated by the infrastructure to a large extent.

This Implementation - Project Skye

  • Platform agnostic - mixed language lemmings
  • No assumptions as to network topology
  • No peer discovery requirement
  • No network size limitation - 1 to millions of lemmings

Plus

Task delegation and load balancing. We have a leader, so why not?

Only 1 Requirement

Some communications infrastructure, like:

  • ESB
  • Publish/Subscribe thing

Skye Algorithm

  • Each lemming chooses a name
  • Seeks out a leader
  • Initiates election when no leader found
  • Election is won by lemming with highest random number between 0 and 1000
